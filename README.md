# Portfólio Isabelle Fujioka

Site estático em HTML, CSS e JavaScript, pronto para publicação no GitHub Pages.

## Como publicar
1. Crie um repositório no GitHub.
2. Faça upload do arquivo `index.html` na raiz do repositório.
3. Vá em Settings > Pages.
4. Em Build and deployment, escolha Deploy from a branch.
5. Selecione Branch: main e Folder: /root.
6. Salve e aguarde o link do GitHub Pages.
